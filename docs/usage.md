 # Usage

 1) Install dependencies: `pip install -e ".[dev]"`.
 2) Run an example server from `examples/`: `python -m http.server 8000`.
 3) In another shell: `politescrape run examples/config.yaml`.
 4) Inspect outputs at `output/example.jsonl` and checkpoints in `.politescrape/checkpoints.db`.

 ## Config fields
 See `examples/config.yaml` and README for detailed options.
